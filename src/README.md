# ShipFlow

## Project Description
ShipFlow is a Java application that processes different shipment types using object-oriented programming principles and the Template Method design pattern.

The system supports:
- Domestic courier shipments
- Pickup point shipments
- International shipments

Each shipment type has its own pricing rules, validation rules, and discounts.

---

## Technologies Used
- Java
- IntelliJ IDEA
- Git
- GitHub

---

## Features
- Abstract base class for shared shipment logic
- Interface for printable summaries
- Method overriding
- Template Method pattern
- Validation rules
- Discount handling
- Different shipment processing strategies

---

## Classes
- ShipmentOrder
- DomesticCourierShipment

- InternationalShipment
- SummaryPrintable

---

## How to Run
1. Open project in IntelliJ IDEA
2. Run Main.java
3. Shipment summaries will be displayed in the console
